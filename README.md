# Elliot

Elliot is a library for adding authentication to [Hanami](http://hanamirb.org) with [Warden](https://github.com/hassox/warden).

## Installation

Add this line to your application's `Gemfile`:

~~~ruby
gem 'elliot'
~~~

And then execute:

~~~
$ bundle install
~~~
